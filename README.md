# Personal Expense Analytics 💰

## Overview

Personal Expense Analytics is a Python-based data analysis project that helps users track and analyze their expenses. The project categorizes spending, calculates key statistics, and visualizes expense distribution using charts.

This project helps beginners understand data analysis, CSV file handling, data visualization, and Python libraries such as Pandas and Matplotlib.

## Features

* 💵 Calculate total expenses
* 📊 Category-wise expense analysis
* 📈 Average expense calculation
* 🏆 Identify highest spending category
* 📉 Bar chart visualization
* 🥧 Pie chart visualization
* ⚡ Beginner-friendly Python project

## Technologies Used

* Python 3
* Pandas
* Matplotlib

## How to Run

1. Install Python 3.
2. Download or clone this repository.
3. Open the project folder in VS Code.
4. Open Terminal.
5. Install required libraries:

```bash
pip install pandas matplotlib
```

6. Run the project:

```bash
python expense_analysis.py
```

## Dataset Columns

| Column   | Description      |
| -------- | ---------------- |
| Date     | Expense date     |
| Category | Expense category |
| Amount   | Expense amount   |

## How It Works

1. Read expense data from a CSV file.
2. Calculate total expenses.
3. Calculate average expenses.
4. Group expenses by category.
5. Find the highest spending category.
6. Generate bar chart and pie chart visualizations.
7. Display results in the terminal.
